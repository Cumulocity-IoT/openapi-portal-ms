Get events V2 (filter + projection)

Return events in the given time range. Supports a safe filter expression (using filtrex) and an optional comma-separated list of fields to project from event.data into the response.

query Parameters
start
required
string <date-time>

Start time as an ISO 8601 date-time string

end
required
string <date-time>

End time as an ISO 8601 date-time string

tenantId
required
string

Tenant identifier (required)

filter
string
Example: filter=name == "loadWidget"

A safe filtrex expression evaluated against each event. You can use the following operations:
